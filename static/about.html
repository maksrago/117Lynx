About

History

Mewch or ChanPink is an English speaking realtime imageboard. Originally created for students on college campuses, this site is used by many. Mewch was made on August 9th 2017. We offer board creation for our users as well as a Radio server, Public File Server and a great amount of support and user interaction.

Features

Mewch uses Lynxchan similar to endchan.
