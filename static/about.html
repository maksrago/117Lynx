About

Mewch (mewch.net) or ChanPink is an English speaking realtime imageboard. Originally created for students on college campuses, this site can be used by anybody.

History

Mewch was made in 2017 by Mega Milk for the purpose of creating a social imageboard. The admin later changed the main board /campus/ into a “Random” type board. The site was posted on 4chan’s /soc/ board many times until the site name became a registered banned url.

Features

Mewch uses Lynxchan similar to endchan.
